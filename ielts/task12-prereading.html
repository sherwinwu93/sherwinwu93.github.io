  • 虚拟语气
  • 特殊句型(强调etc,主语从句)
  • 特殊结构(needn't have done + should have done etc.)

    • 写作任务回应情况

      • 衔接手段运用自如，行文连贯

      • 完全灵活且准确地运用丰富多样的语法结构;极少出现轻微错误，且仅属笔误
