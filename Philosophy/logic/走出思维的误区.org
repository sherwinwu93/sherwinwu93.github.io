* 中译本序言
什么是批判性思维?和其他思维方式什么地方异同?
	- 相同:
		开始于好奇.那些事物可靠,哪些方法有效,它们怎么具备这些品质.
		因为好奇,我们去发现,创造和评价,努力去思考
	- 不同:
		不止于好奇.是思维方式和技巧的精华.
		用来,有人给你提建议时,或者要你按照他说的做,这时候批判性思维就能帮助你做决定.

为什么写这个书?
	- 社会有需要
	- 教的这个技能有用
	- 世面上没有这种实用的书

* 1 恰当提问的益处
批判性思维是什么?
	系统性的技能和态度,具体用法就是一系列相关问题
批判性思维用来干嘛的?
	人用来和信息进行互动,对观点进行判断,形成自己的结论
这些问题具体来说怎么用呢?
	先知道有这些问题,知道什么时候问答这些问题,主动去用这些问题
强弱批判性思维指的是什么?
	接不接受对自我观点的批判,接受就强,不接收就弱.
我原来就有的观点为什么也要被批判?
	1. 原来的观点来审视是不是来自别人的灌输
	2. 不因感情喜恶而接受/拒绝观点
	3. 对他人观点,只要不是人身攻击,都值得拿来审视
要怎么样把批判性思维变成自己的东西呢?
	参与进去,要有舍我其谁的勇气.

** 登堂入室
对的时候提出一系列的批判性问题,由此入门

对信息进行系统的估价--需要一整套态度和技能.这些我们建议在一系列批判性问题上.
	1. 认识全套的相互关联的批判性问题.
	2. 知道什么时候问和答
	3. 主动去用这些问题
靠这些批判性问题入门,靠这些问题培养自己独立思考意识.

** 海绵和淘金
对待知识要先海绵后淘金
|      | 海绵              | 淘金              |
| 方式 | 被动阅读          | 提问阅读          |
|      | 专心致志,死记硬背 | 积极参与,积极反驳 |
|      | 面对判断时,完了   |                   |
| 知识 | 知识的获取        | 与知识的互动      |
海绵是一个必不可少的阶段,先得有东西才能够淘金
或者说只有淘金,海绵改成为捞沙

!!!互动
捞沙->淘金
通过材料给的信息,经过自己的提问互动,判断,形成自己的结论.

** 别相信"正确答案"!
人类行为和科学知识不一样,没有"正确答案".但是可以技巧,获得可取而合理的答案.

这种提问也有适用范围,有时候可以用,有时候救人要紧.

** 感情迷网
避免感情迷网的手段,先提问题,后动感情

不因自己的感情影响理智的见解!因为自己的很多观点不够理性,而是来自旁人的灌输.长时间以后,以至于产生了感情.
如果别人的结论和自己相反,就会觉得自己大受打击.

!!!不能因为感情原因而接受或拒绝一种观点.必须虚怀若谷,吐故纳新.

1. 对待自己以前的观点,要仔细审查是不是来自别人的灌输.
2. 不要因为喜恶去判断观点
3. 对待别人不同的观点,只要不是骂人,都值得检视一番

** 弱和强
批判性思维是把双刃剑,如果用在自我辩解,就落入"弱";如果不但用在别人身上,而且用在自己身上,那才叫"强"

辩解自己,排除异己,就"弱".
批判自己,批判其他,就"强".
双刃剑,弱思维比没有更有害!!!
